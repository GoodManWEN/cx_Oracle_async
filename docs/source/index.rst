Welcome to cx_Oracle_async's documentation!
===========================================
**cx_Oracle_async** is a very simple asynchronous wrapper based on cx_Oracle, that allows you to get access to the Oracle database in asyncio programs. Which is easy to use , buy may not the best practice for efficiency concern.

**cx_Oracle_async** is distributed under an open-source :ref:`license <license>`
(the MIT license).
